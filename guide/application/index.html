Application

A emvc application is a web application that declares RESTful routes and handles requests using controllers, models, and views. emvc applications are organized according to a set of conventions, simplifying configuration and initialization.

emvc extends Express in the same way that Express extends Connect. All emvc applications are Express applications, enhanced with the MVC design pattern and RESTful routing, while preserving the power of middleware and support for template engines.
